# ШумВибро Монитор

**Автоматическая система медиамониторинга структурного шума, вибрации и виброизоляции в многоквартирных домах России**

Собирает публикации от жильцов, управляющих компаний (УК), застройщиков, подрядчиков, производителей оборудования и профильных СМИ.

---

## Возможности

- Еженедельный автоматический сбор данных
- Поиск по новостям, блогам, форумам, VK и сайтам
- Хранение всей истории в SQLite (без дубликатов)
- Выгрузка в **Excel** с удобными колонками
- Полностью редактируемые фильтры (ключевые слова + минус-слова)
- Ручной и автоматический запуск
- Выгрузка за неделю или за последний год
- Фильтрация по региону и типу источника

## Колонки в Excel

- **Дата публикации**
- **Источник**
- **Ссылка**
- **Заголовок + краткая сводка**
- **Регион или город**
- **Ключевые слова, по которым найдено**
